# Countries API Data Project

## Description
This project uses the REST Countries API to collect country information and organize it into a clean dataset. The project practices making API calls, working with nested JSON data, extracting useful fields, storing the data in a pandas DataFrame, and exporting the final dataset as CSV and JSON files.

## Project Goals
- Practice calling a public API using Python
- Understand how to work with nested JSON responses
- Extract specific country details from API data
- Store the extracted data in a pandas DataFrame
- Export the cleaned dataset as CSV and JSON files

## Data Collected
The project collects information such as:

- Country name
- Native name
- Capital
- Population
- Language
- Region

## Tools Used
- Python
- requests
- pandas
- REST Countries API
- VS Code
- Jupyter Notebook

## Files
- `countries_api_project.ipynb` - Main notebook for calling the API and creating the dataset
- `countries_clean.csv` - Cleaned country dataset in CSV format
- `countries_clean.json` - Cleaned country dataset in JSON format
- `README.md` - Project overview and documentation

## What I Learned
- How to make GET requests to an API
- How to check response status codes
- How to navigate nested JSON data
- How to handle fields with changing dictionary keys
- How to convert API data into a structured DataFrame
- How to export cleaned data into CSV and JSON formats
